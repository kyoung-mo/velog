<h3 id="16장-메모리-동적-할당---연습문제">16장 메모리 동적 할당 - 연습문제</h3>
<p>소수(prime number) 계산 프로그램
문제)</p>
<ul>
<li>키보드로 양수를 입력한 후에 입력한 수까지의 모든 소수를 출력합니다. 2부터 한 줄에 5개씩 출력하며 소수가 아닌 수는 X를 출력합니다. 입력한 수에 따라 적절한 크기의 동적 할당하여 사용합니다.</li>
</ul>
<p>실행결과)</p>
<pre><code class="language-bash">양수 입력 : 30
2    3    X    5    X
7    X    X    X   11
X   13    X    X    X
17    X   19    X    X
X   23    X    X    X
X    X   29</code></pre>
<h3 id="17장-사용자-정의-자료형---연습문제">17장 사용자 정의 자료형 - 연습문제</h3>
<p>성적 처리 프로그램
문제)</p>
<ul>
<li>학생 5명의 국어, 영어, 수학 점수를 입력하여 총점, 평균, 학점을 구한다.
학점을 구한 후, 총점 순으로 정렬하여 출력한다.
학점은 평균이 90이상 -&gt; A, 80이상 -&gt; B, 70이상 -&gt;C,
그 외에는 F로 평가한다.</li>
</ul>
<p>실행결과(예시)</p>
<pre><code class="language-bash">학생[1] 국어, 영어, 수학 성적 입력 : 60 60 60
학생[2] 국어, 영어, 수학 성적 입력 : 70 70 70
학생[3] 국어, 영어, 수학 성적 입력 : 80 80 80
학생[4] 국어, 영어, 수학 성적 입력 : 90 90 90
학생[5] 국어, 영어, 수학 성적 입력 : 100 100 100

학생    국어    영어    수학    총점    평균    학점
[5]     100     100     100     300     100     A
[4]     90      90      90      270     90      A
[3]     80      80      80      240     80      B
[2]     70      70      70      210     70      C
[1]     60      60      60      180     60      F</code></pre>

    printf(&quot;두 번째 사람의 나이와 키 : %d, %.1lf\n&quot;, age2, h2);

void swap(void* v1, void* v2, int type) {
    int temp_int;
    double temp_double;

    switch (type) {
    case 4:
        // type int
        temp_int = *(int*)v1;
        *(int*)v1 = *(int*)v2;
        *(int*)v2 = temp_int;

        break;

    case 8:
        //type double
        temp_double = *(double*)v1;
        *(double*)v1 = *(double*)v2;
        *(double*)v2 = temp_double;

        break;
    default:
        break;
    }
}</code></pre>
